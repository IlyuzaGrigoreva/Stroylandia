# Stroylandiya
## Структура проекта:
```
├── common - фрэймворк для работы с системой
│   ├── checkers - место, где лежат все чекеры
│   │   └── __init__.py - для модуля
│   ├── clients - место, где лежат клиенты
│   │   └──  __init__.py - для модуля
│   ├── utils - место-хламосборник, где лежит все вспомогательное
│   │   └──  __init__.py - для модуля
│   └── __init__.py - для модуля
├── etc - конфиги
│   ├── __init__.py - для модуля
│   └── config.py - конфиг
├── tests - папка с тестами и всем, для тестов
│   ├── fixtures - фикстуры для тестов
│   │   └── __init__.py - для модуля
│   ├── my_project - папка с тестами
│   │   └── __init__.py - для модуля
│   ├── __init__.py - для модуля
│   └── conftest.py - место, где делаем длступными фикстуры из fixtures
├── .flake8 - файл с найстройками для flake8
├── .gitignore - файл распеределяющий какой мусор стоит игнорировать, а какой нет
├── .pre-commit-config.yaml - файл с найтроками для pre-commit
├── Dockerfile - файл для docker'а
├── Makefile - файл с make командами
├── poetry.lock - файл лока зависимостей проекта
├── poetry_requirements.txt - версия пакета poetry
├── pyproject.toml - файл с зависимостями и настройками проекта
└── README.md - описание проекта
